# Welcome to The AI Restaurant Manager Kit!

**Tested daily at Brickhouse restaurant** - This system works.

## Quick Start (5 Minutes)

### 1. What You Need
- Raspberry Pi 5 ($50-80)
- Power cable + SD card (32GB+)
- Internet connection
- Toast POS account (for sales data)
- Gmail account (for invoice tracking)

### 2. Install OpenClaw
```bash
# On your Pi 5
npm install -g openclaw
openclaw gateway start
```

### 3. Connect Your Data
- Configure Toast API credentials
- Set up Gmail OAuth for bill tracking
- Add Telegram bot for daily briefings

### 4. Run Your First Briefing
```bash
./Scripts/daily-brief.sh
```
