- Go to: https://gumroad.com/

### 4. Product Files (Zip)

Create a zip containing:
- /setup/ (installation guide)
- /scripts/ (all automation scripts)
- /templates/ (checklists, schedules)
- /tutorials/ (video links)
- /config/ (example configs)
- README.md

### 5. Upload Preview Files

Upload these as "preview" files:
- README.md (customers can read before buying)
- TESTED_FEATURES.md (proves it's real)
- One-page setup guide (PDF)

### 6. Affiliate Program

Enable affiliates:
- 30% commission
- Easy share links
- Gumroad handles payouts

### 7. Email Capture

Enable email list:
- Customers join your newsletter
- Update them on new features
- Gumroad handles opt-ins
